Полноценный интернет-магазин для вашего блога.

== Description ==

[Мерчиум](http://www.merchium.ru) — это сервис для создания полнофункциональных интернет-магазинов, [запущенный совместно с Яндекс.Деньгами](https://money.yandex.ru/doc.xml?id=526675).

Открытие магазина займет всего несколько секунд: просто установите плагин Онлайн-магазин Мерчиум для WordPress и следуйте простым инструкциям по настройке.

= Большие возможности =

Мерчиум дает продавцам множество инструментов для создания прибыльных онлайн-магазинов, от гибкой системы скидок и промо-акций до редактирования товаров прямо на витрине.

Мерчиум создан специально для Рунета и интегрирован с Яндекс.Деньгами, Яндекс.Маркетом (включая продажу товаров прямо в Яндекс.Маркете), 1С, Почтой России и многими другими российскими сервисам.

= Создайте собственный дизайн без знания HTML и CSS =

С помощью встроенного визуального редактора тем вы сможете настроить цветовую схему, логотип и шрифты в своей витрине.

Редактор макетов позволит расположить блоки на странице оптимальным образом.

= Больше тем и приложений в Маркете Мерчиума =

В [Маркете Мерчиума](http://www.merchium.ru/market) вы найдете темы для витрины и приложения, расширяющие функциональность вашего магазина.

= Адаптивный дизайн =

Витрина Мерчиума автоматически адаптируется под размер экрана пользователя и выглядит отлично на экранах любого размера, от мобильных телефонов до больших мониторов.

== Installation ==

= Автоматическая установка =

1. В панели управления WordPress перейдите на страницу **Плагины** (Plugins)
1. Нажмите на кнопку **Добавить новый** (Add New)
1. Наберите в поле поиска *MerchiumRU* и нажмите клавишу Enter
1. Нажмите на кнопку **Установить** (Install Now) под пунктом **Онлайн-магазин Мерчиум**
1. После установки плагина нажмите на ссылку **Активировать плагин** (Activate Plugin)
1. В своей панели управления WordPress перейдите на страницу **Мерчиум** (Merchium Store) и следуйте инструкции по настройке

= Ручная установка =

1. Скачайте плагин на этой странице
1. В своей панели управления WordPress перейдите на страницу **Плагины** (Plugins)
1. Нажмите на кнопку **Добавить новый** (Add New)
1. Нажмите на кнопку **Загрузить плагин** (Upload)
1. Нажмите на кнопку **Browse...** и выберите скаченный архив с плагином (файл в формате .zip)
1. Нажмите на кнопку **Установить** (Install Now)
1. После установки плагина нажмите на ссылку **Активировать плагин** (Activate Plugin)
1. В своей панели управления WordPress перейдите на страницу **Мерчиум** (Merchium Store) и следуйте инструкции по настройке

== Frequently Asked Questions ==

= Могу ли я использовать Мерчиум бесплатно? =

Да, если в вашем магазине 20 товаров или меньше. Платную версию можно бесплатно использовать в течение 14 дней.
